# AI_Informed_Search_DynamicPathFinder
# Dynamic Pathfinding Agent

## Brief Introduction
This project implements a **Dynamic Pathfinding Agent** that navigates a grid-based environment using **informed search algorithms**.  
The agent can dynamically re-plan its path when new obstacles appear while moving. The system includes a graphical interface for visualization and interaction.

Algorithms implemented:
- A* Search
- Greedy Best-First Search (GBFS)

Heuristics supported:
- Manhattan Distance
- Euclidean Distance
